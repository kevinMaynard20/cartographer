# Cartographer Plugin

This plugin provides the `/cartographer` command for Claude Code.

## Commands

- `/cartographer` - Analyze and document the current codebase

## Skills

The cartographer skill enables Claude to automatically map codebases when users request architecture documentation or codebase analysis.

## Triggers

- "map this codebase"
- "analyze architecture"
- "document codebase"
- "create codebase map"
